# Neofolio

Story-driven portfolio for Abhilash Maurya, built with Next.js App Router.

## Stack

- Next.js 14 + TypeScript
- Tailwind CSS + custom design tokens
- Framer Motion + GSAP ScrollTrigger
- React Three Fiber / Drei for ambient 3D visuals

## Run

```bash
npm install
npm run dev
```

## Deploy

Optimized for Vercel default deployment.
